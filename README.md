# 🧮 Calculator CLI App

A simple and user-friendly **Command Line Interface (CLI) Calculator** built with **Python**. This application performs basic arithmetic operations through a menu-driven interface and continues running until the user chooses to exit.

---

## 📌 Objective

The objective of this project is to build a command-line calculator that performs basic arithmetic operations while demonstrating the use of:

- Functions
- Loops
- Conditional Statements
- Exception Handling
- User Input (`input()`)
- CLI (Command Line Interface) Interaction

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔄 Menu-driven interface
- ✅ Input validation
- ⚠️ Division by zero handling
- 🔁 Continuous execution until the user exits
- 💻 Simple, clean, and easy-to-use interface

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash

```

Or download the project as a ZIP file.

### 2️⃣ Open the project folder

```bash
cd Calculator-CLI
```

### 3️⃣ Run the application

```bash
python calculator.py
```
